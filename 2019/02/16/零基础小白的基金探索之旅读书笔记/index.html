零基础小白的基金探索之旅读书笔记

发表于 2019-02-16 | 分类于 学习
零基础小白的基金探索之旅读书笔记

第一章：理财是一件迫在眉睫的事情

  1. 如果仅仅是把多余的钱拿来存银行定期的话，基本上没有办法跑赢通货膨胀。换句话说，存在银行里的钱其实是在贬值的。不过，还是得存钱，这是理财的第一要义，因为存钱可以让自己积攒投资的第一桶金，顺便养成良好的消费习惯。另外银行的定存几乎是无风险的，所以个人或者家庭的一小部分还得存着，作为家庭资产的最后一道防线。

  2. 仅仅靠工资，想要财务自由几乎是不可能的。财富自由这个概念，不同的人有不同的认识和观点，在不工作的前提下也能通过投资等方式获得足够的收入，能有足够的时间做自己想做的事情，更多的和家人在一起，偶尔来一场说走就走的旅行……难道世界上还有比这更美好的事情吗？

    如果想靠工资挣到足够多的钱的话，除非能很顺利的做到公司的高管或者管理层，获得高额的年薪，不过这可能也意味着当个空中飞人，一周也没有时间和家人吃一顿饭，连续几年都放弃年假，这还算好的，因为毕竟有很棒的收入，但是这也只是幸运的少数人。大部分人的工作轨迹却是平平淡淡的，或许 一辈子都是普通员，拿着普通的薪水，还得养家糊口呢，或者在公司有个不 错的职位，拿着还算满意的薪水，过着看起来很美好的中产阶级的生活，但是 一旦没有工作，生活质量就得大打折扣。兔子觉得自己和老公当啥高管的可能性几乎没有了，如果想要在一定程度上摆脱对工作和工资的依赖，唯一的途径 也就只能靠从年轻的时候理财投资了。

  3. 即便抛开财务自由这个美好的概念，就算是为了以后孩子的教育、长辈的赡养、退休以后的晚年生活，理财投资也是必须的！在金钱能发挥作用的地 方，它绝对是非常重要的。难道孩子的奶粉尿布、从幼儿园到大学的学费、父母的赡养费、退休后的养老金能从天上掉下来不成？难道一句“车到山前必有 路”就能变出钱来？

  4. 理财投资绝对是门学问。

第二章：初识基金

货币基金

货币基金优势：基金经理会把投入基金的钱投资到风险小的短期货币工具上，比如国债啊、央行票据、商业票据、信用等级较高的企业债券等。

货币基金特点：风险低收益也就不会太高，但至少比活期存款高很多。

债券基金

定义：主要投资债券的基金，一般债券占投资 80%以上的基金就可以叫做债券基金了。主要是投资于国债、地方债、金融债、公司债等等，当然还有一小部分钱也可以投 资到股票市场和可转债市场中。有些债券品种是很难以个人投资者的身份接触 到的，债基相当于给咱们一个机会来参与债券市场。其实再简单点说就是基金经理拿咱们的钱借给政府、银行或者企业，定期收取利息，到期再收回本金。

优点：债基的收益还是比较稳定。另外因为投资的债券时间比较长，种类比较多，有时候还可以靠投资点股票什么的获取点额外收益，所以整体来说收益是比货币基金稍微高一 些。

缺点：灵活性没有货币基金高。

货币基金 or 债券基金

  1. 流通性：货基投资的是短期的货币产品，一般来说平均期限为 120 天，不会超过一年，这就意味着货币基金的流动性比较好。

    债基投资的产品期限可能是一年以上甚至更长，要知道对于债券来说，一般是时间越长收益越高。

  2. 风险性：债基投资的产品风险相对货基来说会更高一些，以公司债为例，有些债基为了获取较高的收益，可能会投资些信用等级比较低的公司债，就存在公司破产还不出钱的情况发生。

    货基的资金投资更谨慎，一般会选择信用等级比较高的企业债。

股票基金

定义：股票基金就是把钱交给基金经理，让基金经理帮咱们买股票。

优点：因为一只股票基金肯定会持有多只股票，这样也在一定程度上可以降低风险。不同的股票基金持有的债券比例不同，购买的股票不同，因此风险和收益也会有所不同。”

指数基金

定义：指数基金就是跟踪一个特定的证券指数的基金，比如广发沪深 300 指数，就是跟踪沪深 300 指数的指数基金。

优点：管理费特别便宜。

第三章：如何找到好基金

九字真言：风险性、收益性和流动性。

所谓流动性，就是指资产能够以一个合理的价格顺利变现的能力，越快的话流动性越好。

在选择基金的时候就要结合自己的需求和基金的特点来通盘考虑，不能只看收益率，要知道收益越高风险也就越高。

标准差：衡量基金增长率波动情况的指标，越大的话说明基金收益变化越大。

夏普比率：综合了收益和风险的系数，如果两只收益一致的基金，夏普比率高的那一只说明风险较低（备注：夏普比率一定要在同类基金中做比较）

阿尔法系数：代表基金多大程度能跑赢市场，当然越高越好啦。

贝塔系数：基金相对于大盘的波动情况。

R 平方：代表基金和大盘的相关性。基金和大盘越相关，贝塔系数的参考价值就越大。

第四章：适合的才是最好的

如何选择 最适合自己的基金吗？

要选择合适的基金，不仅仅要明确自己的投资目标，还需要了解自己的风险承受能力和态度，这样才能有的放矢，真正找到适合自己的。

晨读感悟:
如果可以实现财务自由,就可以有更多的时间自己追求内心真正最求的东西,也会有更多的时间陪伴家人。
希望可以通过训练营提升自己的理财认知。
